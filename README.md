#### Stream流编程
这是JDK1.8提供的流式编程，主要简化了操作集合、Map，以及一些更高级的用法

#### Lambda
Lambda表达式在很多后端语言中，很常见，常见的c++、python都支持Lambda表达式，通常该表达式能够解决复杂的调用链以及更简化的操作方法等，能够使代码更加简洁和优雅
